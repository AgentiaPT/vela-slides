# Security Policy

For the full security model, threat surface analysis, and bounty program, see **[docs/SECURITY.md](docs/SECURITY.md)**.

## Reporting Vulnerabilities

1. **Do not** open a public GitHub issue
2. Email: **info@agentia.pt**
3. Include: description, reproduction steps, and potential impact
4. We will acknowledge receipt within 48 hours

## Bounty Program

We offer symbolic bounties ($10-$50) for valid security findings. See [docs/SECURITY.md](docs/SECURITY.md#security-bounty-program) for tiers, rules, and scope.
